## 興味・関心

どのような会社に興味・関心があるか

### 事業

- 社会問題の解決に向き合うもの
- 地方の活性化、発展に寄与するもの
- 新たな領域へ挑戦するもの

### 会社・組織

- 明確なミッションがある
- チームで働くこと、コミュニケーションを重要視している
- リモートを前提とした組織づくりをしている
- 新しいことへの挑戦を許容している
- 全員がよりよいエンジニアリングを追求している
- 技術への関心とリスペクトがある
- 少人数のチームで改善プロセスをまわすアジャイルスタイルである
- 部署、技術レイヤーをこえて協力するスタイルの組織である
